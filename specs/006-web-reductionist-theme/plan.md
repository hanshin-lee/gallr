# Implementation Plan: Web Reductionist Design System

**Branch**: `006-web-reductionist-theme` | **Date**: 2026-03-20 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/006-web-reductionist-theme/spec.md`

## Summary

Port the reductionist design system from the KMP app (005) to the gallr web presentation site. Three CSS-only changes: (1) replace Playfair Display + JetBrains Mono with Inter via `@fontsource/inter`; (2) add `--color-accent: #FF5400` and apply it to primary CTA buttons; (3) remove the thick decorative `border-top` separating page sections. No HTML structure changes. No new JavaScript. No backend changes.

## Technical Context

**Language/Version**: HTML5 / CSS3, JavaScript ES2022 (build scripts only — no runtime JS)
**Primary Dependencies**: Eleventy 3.x (static site generator), `@fontsource/inter` (replace `@fontsource/playfair-display` + `@fontsource/jetbrains-mono`), pa11y 8.x (accessibility), Playwright 1.49 (visual acceptance tests)
**Storage**: N/A — fully static
**Testing**: pa11y (accessibility audit) + Playwright (visual regression); existing test infrastructure in `web/tests/`
**Target Platform**: Modern browsers — Chrome 111+, Safari 16.2+, Firefox 113+ (all support `color-mix()`)
**Project Type**: Static marketing website (Eleventy, output to `web/dist/`)
**Performance Goals**: `font-display: swap` — ≤2s FCP; Inter WOFF2 ≈ 60KB per weight (smaller than Playfair Display 700 ≈ 89KB)
**Constraints**: CSS-only changes; no HTML structure changes; zero JS logic changes; `color-mix()` requires browser support (Chrome 111+, Safari 16.2+, Firefox 113+) — all modern; falls back gracefully
**Scale/Scope**: Single marketing homepage + coming-soon page; 6 CSS files total, ~400 lines of CSS affected
